# ✅ Habitly — Habit Tracker

Build routines by making them visible. Add habits, tick each day of the week, and track your current streak and total days. Everything saved locally in your browser.

**Live demo:** https://skrizidev.github.io/habit-tracker/

## Pages
- **Today** — this week's grid with tappable day boxes (today highlighted)
- **Stats** — current streak and total completions per habit
- **About** — the "don't break the chain" idea

## Tech
Vanilla HTML / CSS / JavaScript. State in `localStorage`, Monday-based weeks.
